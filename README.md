■ 家計簿アプリ（サンプル README）
■ 概要

React を用いて作成したシンプルな家計簿アプリです。
日々の支出・収入を入力し、カテゴリ別に管理できるよう実装しています。
月ごとの合計金額・カテゴリ集計を確認でき、シンプルで使いやすい家計簿を目指しています。

■ 使用技術

React

JavaScript

HTML / CSS

Vite

■ 主な機能

支出／収入の登録（金額・カテゴリ・メモ）

登録した一覧の表示・削除

カテゴリ別集計（食費・日用品・交通費など）

月ごとの合計金額表示

ローカルストレージによるデータ保持

■ 起動方法
npm install
npm run dev
